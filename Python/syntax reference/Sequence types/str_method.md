# str method
  
## replace
  
	문자열.replcae("바꿀 문자열", "새 문자열")
  
해당 문자열 내에 있는 문자열을 인수로 전달된 새 문자열로 바꾼다.  
  
## maketrans
  
	문자열.maketrans("바꿀 문자", "새 문자")
  
문자열 속에 있는 해당 문자를 특정한 값으로 변경할 수 있는 테이블을 만든다.  
  
## translate
  
	문자열.translate(테이블)
  
maketrans로 만든 테이블을 활용하여 문자열 안의 문자를 다른 문자로 바꾼다.  
  
## split
  
	문자열.split("기준 문자열")
  
기준 문자열을 기준으로 문자열들을 나누어서 해당 문자열들을 묶어 리스트로 만든다. 기준 문자열을 생략하면 기본적으로 공백이 기준 문자가 된다.  
  
## join
  
	문자열.join(리스트)
  
문자열을 문자열 리스트 혹은 튜플의 요소와 연결하여 문자열로 만든다.  
  
## upper
  
	문자열.upper()
  
문자열 내에 있는 소문자를 모두 대문자로 바꾼다.  
  
## lower
  
	문자열.lower()
  
문자열 내에 있는 대문자를 모두 소문자로 바꾼다.  
  
## strip, lstrip, rstrip
  
	문자열.strip("삭제할 문자열")
  
strip은 문자열의 양쪽에서 삭제할 문자열들을 문자열에서 삭제한다. 삭제할 문자열을 지정하지 않을 경우 공백 문자가 삭제된다.  
  
	문자열.lstrip("삭제할 문자열")
  
문자열의 왼쪽에서만 문자를 삭제하고자 한다면, lstrip을 사용하면 된다. 이 함수는 문자열의 왼쪽에서 삭제할 문자열들을 삭제한다. 인수를 지정하지 않을 경우 공백 문자가 삭제된다.  
  
	문자열.rstrip("삭제할 문자열")
  
문자열의 오른쪽에서 삭제할 문자열들을 삭제할 수 있다. 삭제할 문자열을 지정하지 않으면 공백 문자가 삭제된다.  
  
## ljust, rjust
  
	문자열.ljust(길이)
  
인수에 전달된 길이만큼 메모리 공간을 확보한 뒤, 왼쪽에서부터 문자열을 할당하기 시작한다. 남은 공간은 모두 공백으로 넣어진다. 왼쪽 정렬이라고 생각하면 된다.  
  
	문자열.rjust(길이)
  
인수에 전달된 길이만큼 메모리 공간을 확보한 뒤, 오른쪽에서부터 문자열을 할당하기 시작한다. 물론 문자열은 거꾸로 들어가지 않고 제대로 들어간다. 남은 공간은 공백으로 채운다. 오른쪽 정렬이라고 생각하면 된다.  
  
## center
  
	문자열.center(길이)
  
인수에 전달된 길이만큼 메모리 공간을 확보한 뒤, 가운데 정렬을 한다. 남은 공간은 공백으로 채워진다. 보통 남은 공간은 (길이 - 문자열의 길이) / 2로 왼쪽, 오른쪽에 공평하게 남게 되나, 만약 2로 나누어떨어지지 않는다면 왼쪽 공간에 남는 공간이 1개 더 추가된다.  
  
## zfill
  
	문자열.zfill(길이)
  
지정된 길이만큼의 메모리 공간을 확보한 뒤, 해당 문자열을 오른쪽 정렬한다. 그 뒤, 남은 공간에는 0을 넣는다.  
  
## find, rfind
  
	문자열.find("찾을 문자열")
  
문자열의 왼쪽에서부터 특정 문자열을 찾아서 해당 요소의 인덱스를 반환한다. 만약 특정 문자열이 해당 문자열에 존재하지 않는다면 -1을 반환한다.  
  
	문자열.rfind("찾을 문자열")
  
find와 기능은 동일하나 찾는 순서가 오른쪽에서부터 찾기 시작한다.  
  
## index, rindex
  
	문자열.index("찾을 문자열")
  
문자열의 왼쪽에서부터 특정 문자열을 찾아서 해당 요소의 인덱스를 반환한다. find와 기능은 동일하지만 찾고자 하는 문자열이 존재하지 않는다면 에러가 발생한다는 점이 다르다.  
  
	문자열.rindex("찾을 문자열")
  
index와 기능은 동일하나 찾는 방향이 오른쪽에서부터 찾기 시작한다는 점이 다르다.  
  
## count
  
	문자열.count("문자열")
  
현재 문자열에서 특정 문자열이 몇 번 등장하는지 알아내며 해당 개수를 반환한다.
