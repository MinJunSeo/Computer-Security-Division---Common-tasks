# Study topic
  
Node.js - 사용자에게 입력 받기 & 서버로 데이터 전송하기  
  
# Goal
  
1. HTML 태그를 활용하여 사용자에게서 데이터를 입력받을 수 있다.  
  
2. Node.js 서버로 데이터를 올바르게 전송할 수 있다.  
  
3. 서버로 보낸 데이터가 잘 전송되었는지 확인할 수 있다.  
  
4. POST 방식으로 보낸 데이터를 코드 내로 가져와 활용할 수 있다.
